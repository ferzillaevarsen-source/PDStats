Импорт истории

Вставьте историю турниров из личного кабинета ПокерДом

Как скопировать:

  1. Войдите в Личный кабинет → История → Турниры
  2. Задайте период, выделите всё Ctrl+A, скопируйте Ctrl+C
  3. Вставьте сюда Ctrl+V

Реентри (#2, #3...) объединяются в одну строку с суммарным взносом. Лиговые турниры определяются по названию.

Авто-импорт: запустите pdhelper.py, откройте историю турниров во вкладке Турнир в PokerDom, нажмите F1 — данные появятся здесь автоматически.
Установка: python -m pip install pywin32 keyboard pystray Pillow
— иначе токен хранится только до закрытия вкладки
Для безопасности используйте fine-grained токен с доступом только к этому репозиторию и правом Contents: Read-only. Классический ghp_ даёт доступ ко всем вашим репозиториям — не рекомендуется.
Облачный канал: токен сохранён (только на сессию)
